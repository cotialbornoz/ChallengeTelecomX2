# **Challenge Telecom X Parte 2**

Trabajo final del Challenge Alura Store del curso de Estadisticas y Machine Learning para Data Science de ALURA Latam | ONE Oracle Next Project. Se basa en el concepto Challenge Based Learning, es decir, un aprendizaje centrado en desafíos que Apple ayudó a crear y que se basa en 3 pilares: Compromiso - Investigación - Actuación.

Con esto, se buscó poner en práctica habilidades esenciales de análisis de datos en un escenario de negocios real, construyendo modelos predictivos capaces de prever qué clientes tienen mayor probabilidad de cancelar los servicios de la empresa contratista.

Proyecto **finalizado**.

## **Objetivo del análisis realizado**

Se buscó brindale a la empresa Telecom X, a través del proyecto "Churn de Clientes", un análisis detallado de los factores que llevan a la pérdida de clientes, con el objetivo de bajar la alta tasa de cancelaciones.

Se buscó brindarle a la empresa Telecom X, a través de la 2da parte del proyecto "Churn de Clientes", un modelo predictivo capaz de prever qué clientes tienen mayor probabilidad de cancelar sus servicios.

## **Proyecto**

### Estructura

El proyecto se compone de 4 partes:

1. Preparación de los Datos: se eliminaron las columnas irrelevantes para el análisis, a través de matriz de correlación y Chi al cuadrado. Además, se transformaron las variables categóricas a formato numérico para hacerlas compatibles con los algoritmos de machine learning. 
2. Correlación y selección de variables: se visualizaron las relaciones entre las variables con matrices de correlación, con el fin de prestar atención a las variables que muestran una mayor correlación con la cancelación. 
3. Modelo predictivo: tras la separación de los datos en variables explicativas y la variable objetiva, se crearon dos tipos de modelos, modelo Árbol de Decisión y modelo K-vecinos más cercanos. Estos modelos fueron entrenados, evaluados y refinados.
4. Interpretación y conclusiones: se analizaron las 10 variables más importantes por modelo para así obtener las que más se repetían. En la conclusión final, se brinda propuestas de mejora del servicio para así disminuir la tasa de clientes que se fugaron.
